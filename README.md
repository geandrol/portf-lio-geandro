## 💻 Sobre a Aplicação

Este é o repositório do meu portfólio profissional. Foi desenvolvido com foco em **alta performance, arquitetura limpa e UI/UX moderna**, refletindo o padrão de qualidade exigido no mercado sênior de tecnologia.

A aplicação apresenta uma interface responsiva ("mobile-first"), com suporte a Dark/Light mode, animações fluidas baseadas em scroll e microinterações ("glassmorphism"), proporcionando uma experiência premium.

### 🛠 Tecnologias Utilizadas

- **React 19** (Vite)
- **TypeScript** (Tipagem estática e segurança)
- **Tailwind CSS v4** (Estilização utilitária e Dark Mode)
- **Framer Motion** (Animações de entrada, scroll e hover)
- **Lucide React** (Ícones modernos)

## ⚙️ Como rodar o projeto localmente

Pré-requisitos: Ter o [Node.js](https://nodejs.org/) instalado.

```bash
# 1. Clone este repositório
git clone https://github.com/Marcelo7211/portifoleoCv.git

# 2. Acesse a pasta do projeto
cd portifoleoCv

# 3. Instale as dependências
npm install --legacy-peer-deps

# 4. Inicie o servidor de desenvolvimento
npm run dev
```

Acesse `http://localhost:5173` no seu navegador para ver o resultado.
